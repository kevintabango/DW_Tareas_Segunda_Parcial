Selectores Css

Selectores de Tipo y Clase

Selector de tipo: Aplica estilos a todos los elementos HTML del mismo tipo, por ejemplo p, h1, ul. Ejemplo: p { color: blue; } — Estilo para todos los párrafos. .

Selector de clase: Aplica estilos a los elementos que tienen una clase específica (precedida por un punto). Ejemplo: .resaltado { font-weight: bold; } — Afecta solo a los elementos con class="resaltado"

Selector Descendente

Aplica estilos a los elementos anidados dentro de otro, sin importar cuántos niveles de profundidad haya. Ejemplo: article p { color: green; } — Estilo solo para los p dentro de article..

Selectores de ID y Hijos Directos

Los selectores de ID se utilizan para aplicar estilos a un único elemento con un ID específico.

Ejemplo: #contenedor { background-color: yellow; } — Afecta solo al elemento con id="contenedor-destacado".

Selector de Hijos Directos

Aplica estilos solo a los elementos que son hijos directos de un elemento específico.

Ejemplo: div > p { color: red; } — Afecta solo a los párrafos que son hijos directos de un div.

Pseudoclases y Pseudoelementos

Pseudoclase: Permite aplicar estilos según el estado del elemento. Ejemplo: a:hover { color: red; } — Cambia el color al pasar el mouse.

Pseudoelemento: Permite aplicar estilos a una parte específica del elemento. Ejemplo: p::first-line { font-weight: bold; } — Aplica estilo solo a la primera línea del párrafo.
